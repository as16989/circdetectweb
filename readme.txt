To run the website locally:

1. Clone this repo into a directory of your choosing
2. Install pip
3. Install flask
4. Run 'pip install -r req.txt' whilst in the cloned directory (~/circdetectweb)
5. Run 'flask run'
6. In either Firefox or Chrome, enter localhost:5000 into the address bar

This website has only been tested and verified as working correctly on Firefox and Chrome.
